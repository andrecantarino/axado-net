## axado-net
Cadastro e avalição de transportadoras utilizando as tecnologias .NET (MVC).

### API
Esse projeto consome uma API em NodeJS e banco de dados MongoDB hospedada em: http://apiaxado.andrecantarino.com.br/

### APLICAÇÃO
Na pasta axado está o código da aplicação que irá consumir a API, foi utilizado: .NET

### COMO FUNCIONA?
1. Você deverá baixar o projeto do github: git@github.com:andrecantarino/axado-net.git
2. A API e o banco estão disponíveis online no endereço: http://apiaxado.andrecantarino.com.br
3. Você deverá abrir o arquivo 'axado.sln' e executar o projeto localhost.
4. Depois de executado, acesse o menu LOGIN com os dados passados por e-mail
5. Haverá uma lista de transportadoras cadastradas, onde o usuário poderá efetuar um CRUD básico.
6. O usuário poderá avaliar a tranportadora apenas uma única vez.
